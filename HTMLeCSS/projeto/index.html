O que são Clients?

Clients ou traduzindo para o português Clientes são os browseres. Quando entramos em um site, o navegador faz uma requisição, enviando o protocolo HTTP para o servidor e o servidor devolve em resposta arquivos HTML, CSS, JavaScript, Imagens, etc.

"O lado client é quando você consome a informação de algum lugar baseado na web, o client é o dispositivo que está consumindo aquela informação na internet". Diogo Mainardes
>voltar

O que são Servers?

Servers ou traduzindo para português Servidores. Servidores são locais onde hospedamos os sites. Quando você coloca o seu site na intenet, você vai hospedar ele no servidor para quando alguém digitar no seu site por exemplo www.site.com.br e der um enter, ele vai através da intenet encontrar um computador, onde vai estar o seu arquivo do site e vai enviar essa informação de volta. Esses arquivos de retorno que são arquivos HTML, CSS, JavaScript, Imagens, Áudios e outros são chamados de arquivos estáticos, que o servidor envia para o cliente.

"O lado servidor serve informação para os clientes."Diogo Mainardes

Como é uma estrutura básica de um HTML?

A estrutura básica de um documento HTML, ou seja, de uma página da web, é realizada da seguinte forma: 

      <!DOCTYPE html>
      <html>
      <head>
          <meta charset="utf-8">
          <title>></title>
      </head>
      <body><
      </body>
      </html>

A partir desse código, podemos desenvolver toda a nossa página, estruturando como precisarmos. Mas primeiro, é bom compreender o que está sendo feito até o momento:

  • <!DOCTYPE html> — A declaração do doctype informa ao navegador que está sendo utilizado a linguagem HTML . Hoje em dia, o doctype basicamente é necessário apenas para garantir que o documento se comporte corretamente.

  • <head></head> — O elemento <head> é o local no qual você irá incluir todos os elementos que deseja incluir na página mas que não deve ser renderizado. Dentre eles, podemos utilizar os metadados, o título, as tags <link> para adicionar folhas de estilos, entre outras.

  • <meta charset=”utf-8″> — Esse elemento de metadados serve para definir o conjunto de caracteres que seu documento deve usar para o utf-8.

  • <title></title> — O elemento <title> serve para definir o título de sua página. Portanto, é aqui que se define o título que é mostrado na guia do navegador.

  • <body></body> — Finalmente, neste elemento, ficarão todos os elementos que serão renderizados pelo navegador, ou seja, todo o corpo da sua página HTML.

Entendendo a diferença entre listas ordenadas e não ordenadas

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados.

A lista ordenada enumera os elementos e a lista não-ordenada coloca marcadores antes de cada elemento.

Compare as duas listas a seguir:

Lista ordenada Lista não-ordenada
  1. Item um.
  2. Item dois.
  3. Item três.
  • Primeiro item.
  • Segundo item.
  • Terceiro item.
